# Swarm

**バージョン: v1.5.1**

攻撃的なモブを召喚し、対象プレイヤーを襲わせる**汎用妨害コマンド**プラグインです。TikTok 妨害配信で、配信者へモブ妨害を仕掛ける用途で作成しています。

---

## 概要

指定したモブを対象プレイヤーの周囲に召喚し、そのプレイヤーを攻撃させます。召喚モブは設定/引数で指定した**攻撃回数だけ攻撃すると消滅**します。蜂・狼などの中立モブも怒り状態で召喚されます。

ガチャ等のコンソールコマンドから `/swarm bee 1 1 %player%` のように呼び出すこともできます。

---

## コマンド

```
/swarm <モブ> [攻撃回数] [匹数] [対象]
```

| 引数 | 意味 | 省略時 |
|---|---|---|
| 第1: モブ | `mobs:` エイリアス または EntityType 名（`Mob` 派生のみ可） | `default-mob` |
| 第2: 攻撃回数 | 何回攻撃したら消滅するか | `attack-count`(既定1) |
| 第3: 匹数 | 召喚する数（`max-count` 上限） | `count`(既定1) |
| 第4: 対象 | プレイヤー名 / `all` / `random`（省略時は実行者自身） | 実行者 |

| コマンド | 説明 |
|---|---|
| `/swarm bee` | 蜂を1匹、自分に召喚（既定回数で消滅） |
| `/swarm bee 2 1` | 蜂を1匹、**2回攻撃で消滅** |
| `/swarm zombie 1 10 all` | ゾンビ10体をオンライン全員に |
| `/swarm bee 3 1 <名前>` | 蜂を1匹、指定プレイヤーに（3回攻撃で消滅） |
| `/swarm reset` | **プレイヤー以外の全エンティティを消去**（`/kill @e[type=!player]` 相当・テスト用） |
| `/swarm reload` | `config.yml` を再読み込み |

- 権限: `swarm.use`（デフォルト op）
- タブ補完4段（モブ / 攻撃回数 / 匹数 / 対象）

---

## 仕様詳細

- **召喚**: 対象頭上付近（`spawn-radius` / `spawn-height`）に湧かせ、`setTarget(対象)` で狙わせる。中立モブは Bee=`setAnger`＋`setCannotEnterHiveTicks` / Wolf=`setAngry(true)`。`setPersistent` ＋ `setRemoveWhenFarAway(false)` で消えにくくする。
- **被弾累計で消滅（v1.5.0〜）**: 「同じ個体が N 回刺す」のではなく、**対象が swarm モブから N 回（=攻撃回数）被弾するごとに 1 匹消える**累計方式。例: `/swarm bee 2 5 all` なら 2 回被弾ごとに 1 匹、延べ 10 回で 5 匹全消し。被弾はどの個体からでも累計され、余りは次に持ち越す。召喚モブは PDC（`swarm` タグ＋`swarm_remaining`＝消すのに必要な被弾しきい値）で識別。
  - ダメージが無効化される状況（クリエイティブ無敵・`/barrier` のモブ攻撃無効化中など）でも「接触＝攻撃判定」を数えるため、攻撃イベントの `ignoreCancelled` は付けていない（環境に依らず消える）。
  - ⚠️ サバイバルでは無敵時間（i-frame, 0.5秒）があるため、複数匹が同時に刺さっても被弾カウントは概ね 0.5 秒に 1 回。多数召喚すると全消しまで時間がかかる。
- **reset**: 全ワールドの**プレイヤー以外の全エンティティ**を削除します（タグ有無に依存しない）。
  - ⚠️ ドロップアイテム・矢・防具立てなども巻き込むため**テスト用**。バニラ `/kill @e[type=!player]` と同じ副作用があります。

## config.yml（抜粋）

```yaml
default-mob: bee     # モブ省略時に使うモブ
count: 1             # 1対象あたりの召喚数（第3引数で上書き可）
attack-count: 1      # 消滅までの攻撃回数（第2引数で上書き可）
anger-ticks: 99999   # 中立モブを怒らせておくティック数（20=1秒）
spawn-radius: 3.0    # 召喚位置の水平散らばり半径
spawn-height: 1.5    # 対象足元からの高さ
persistent: true     # 遠くでも消えないように
max-count: 100        # 一度の最大召喚数（負荷対策）

mobs:                # エイリアス（コマンドで打つ名前）: EntityType
  bee: BEE
  zombie: ZOMBIE
  skeleton: SKELETON
  spider: SPIDER
  blaze: BLAZE
  vex: VEX
  wolf: WOLF
  creeper: CREEPER
```

- `mobs:` に無い EntityType 名（例: `phantom`, `enderman`）も直接指定できます（攻撃可能な `Mob` のみ）。

---

## インストール / 反映

1. ビルドした `Swarm.jar` を `plugins/` に配置。
2. サーバー再起動（または `/reload confirm`）で有効化。`config.yml` は初回起動時に自動生成。

ビルドは [minecraft-build-env] のオフライン手順。

|---|---|
| v1.5.1 | `/swarm reset` の実行ログを『モブを消去しました。』に簡略化（削除数・`/kill @e[type=!player]` の注記を省略）。挙動は v1.5.0 と同じ。 |

| v1.4.0 | `/swarm reset` を「プレイヤー以外の全エンティティ削除（`/kill @e[type=!player]` 相当）」に変更。 |
| v1.3.0 | 引数順を `<モブ> [攻撃回数] [匹数] [対象]` に変更。reset の堅牢化（後に v1.4.0 で全消去仕様へ）。 |
| v1.2.0 | 攻撃回数で消滅する仕様（PDC タグ）＋ `/swarm reset` 追加。 |
